## 五、系统流程

### 5.1 用户搜索流程
```
用户访问 www.io011.com (1566)
    ↓
输入搜索词
    ↓
请求 → Hub API (1569) /api/v1/search?tags=xxx
    ↓
Hub 并行查询已注册商户 (最多5个)
    ↓
返回结果给前端
    ↓
显示商户/商品列表
```

### 5.2 商户接入流程
```
商户安装 merchant-skill
    ↓
配置 merchant.yaml (Hub地址)
    ↓
启动 Skill (端口1589)
    ↓
到 Hub 后台注册商户
    ↓
Hub 存储商户 endpoint
    ↓
用户搜索时调用该商户
```

## 🔗 通讯流程

### 1. 用户搜索流程
```
1. 用户在 www.io011.com 输入搜索词
2. 请求发送到 Hub API (api.io011.com)
3. Hub 解析搜索词 → 提取标签
4. Hub 并行查询已注册商户 (最多 5 个，2 秒超时)
5. Hub 聚合结果，返回给用户
```

### 2. 商户接入流程 (A2A)
```
1. 商户下载 OpenClaw 助手
2. 安装 merchant-skill
3. 配置 merchant.yaml (填写商户信息 + Hub 地址)
4. 启动 Skill (生成 A2A 接口)
5. 商户到 Hub 后台注册 (提交 endpoint)
6. Hub 审核通过后，商户即可被搜索到
```

## 🔐 安全机制

1. **API Key 认证**: 每个商户有独立 API Key
2. **HTTPS 强制**: 生产环境全站 HTTPS
3. **限流**: 每个 API 100次/分钟
4. **签名验证**: A2A 通信使用签名

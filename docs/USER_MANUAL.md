## 🛒 用户使用指南

### 1. 访问平台

打开浏览器，访问: **www.io011.com**

### 2. 搜索商品

在搜索框中输入你想要找的商品，例如：
- "茶具"
- "景德镇陶瓷"
- "手工艺术品"

### 3. 查看搜索结果

搜索结果会分步骤显示：

```
Step 1/3: 解析需求 ✓
  正在分析您的搜索词...

Step 2/3: 匹配商户 ✓
  找到 3 家相关商户
  • 老茶铺工作室 [景德镇 | 4.8⭐]
  • 青瓷工坊 [龙泉 | 4.9⭐]

Step 3/3: 获取商品...
  [████████░░] 67%
  已返回 2/3 商户结果
```

### 4. 筛选和排序

在结果页面可以使用筛选器：
- **类目筛选**: 茶具、陶瓷、手工艺术品...
- **地区筛选**: 景德镇、龙泉、宜兴...
- **价格筛选**: 0-50元、50-100元...
- **排序**: 相关度、价格、评分

### 5. 查看商品详情

点击商品卡片查看详情：
- 商品图片
- 价格
- 商品描述
- 所属商户信息

## 🏪 商户使用指南

### 1. 访问商户后台

打开浏览器，访问: **shop.io011.com** (端口 1588)

### 2. 登录

使用商户账号登录（首次使用需注册）

### 3. 商户中心功能

#### 3.1 商品管理
- **商品列表**: 查看所有商品
- **添加商品**: 新增商品
- **编辑商品**: 修改商品信息
- **上下架**: 控制商品销售状态
- **库存管理**: 修改库存数量

#### 3.2 标签管理
设置商户标签，让用户更容易找到你：
- **类目标签**: 茶具、陶瓷、手工艺术品...
- **能力标签**: 支持定制、48小时发货、包邮...
- **地区标签**: 景德镇、龙泉、宜兴...
- **价格区间**: 商品价格范围

#### 3.3 订单管理
- **订单列表**: 查看所有订单
- **订单详情**: 查看订单详细信息
- **发货**: 填写物流信息
- **取消订单**: 处理退款

#### 3.4 数据分析
- **访问统计**: 店铺访问量
- **商品浏览**: 各商品被查看次数
- **销售统计**: 销售额、订单数

## 🔧 商户接入流程 (A2A 模式)

### 步骤 1: 安装 OpenClaw

下载并安装 OpenClaw 助手

### 步骤 2: 安装 merchant-skill

在 OpenClaw 技能市场中搜索 "merchant-skill" 并安装

### 步骤 3: 配置商户信息

编辑 `config/merchant.yaml`:
```yaml
server:
  port: 1589

merchant:
  name: "你的商户名称"
  description: "商户描述"

hubs:
  - name: "AICart Hub"
    url: "https://api.io011.com"  # Hub API 地址
    api_key: "你的API密钥"
    auto_sync: true
```

### 步骤 4: 启动 Skill

```bash
cd merchant-skill
npm install
node src/index.js
```

### 步骤 5: 注册到 Hub

在商户后台完成商户信息注册，提交你的 A2A 接口地址

## ❓ 常见问题

### 用户常见问题

**Q: 搜索不到想要的商品怎么办？**
A: 尝试使用更通用的关键词，或减少筛选条件

**Q: 商品价格在哪里看？**
A: 点击商品卡片，在详情页面查看价格

**Q: 如何联系商户？**
A: 点击商户名称查看商户联系方式

### 商户常见问题

**Q: 为什么我的商品搜不到？**
A: 检查是否已正确设置标签，并确保商品已上架

**Q: 如何修改商品信息？**
A: 在商品管理页面点击"编辑"按钮

**Q: 订单从哪里查看？**
A: 在左侧菜单点击"订单管理"

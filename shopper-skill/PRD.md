# IO011/AICart - 用户端 (io011-shopper) PRD

## 产品概述

**产品名称**: io011-shopper (AI购物车用户端)  
**版本**: MVP v0.1  
**目标用户**: 希望通过自然语言购买商品的普通消费者  
**核心定位**: Agent-to-Agent (A2A) 电商体验的用户入口

---

## 1. 用户故事 (User Stories)

### US-001: 自然语言发起购物请求
**作为** 消费者  
**我希望** 用自然语言描述我想购买的商品（如"想买茶杯，100-200元"）  
**以便** 无需浏览多个平台即可表达购物意图  

**验收标准**:
- 支持中文自然语言输入
- 识别商品类目关键词
- 识别价格区间（支持"元"、"块"、"左右"等表述）
- 响应时间 < 2秒

---

### US-002: AI理解并提取购物意图
**作为** 消费者  
**我希望** AI能准确理解我的需求并确认关键信息  
**以便** 确保搜索结果符合我的预期  

**验收标准**:
- 提取商品类目（如"茶杯"→"茶具/茶杯"）
- 提取价格区间（如"100-200元"→min:100, max:200）
- 向用户确认理解结果（"为您搜索100-200元的茶杯，对吗？"）
- 支持用户修正参数

---

### US-003: 搜索匹配商户
**作为** 消费者  
**我希望** 系统自动查找销售相关商品的商户  
**以便** 我无需手动筛选商户  

**验收标准**:
- 调用Hub标签搜索API
- 根据类目匹配商户标签
- 返回至少3个匹配商户（如有）
- 商户响应时间 < 3秒

---

### US-004: 并行获取商品列表
**作为** 消费者  
**我希望** 同时查看多个商户的商品  
**以便** 比较价格和选择最优选项  

**验收标准**:
- 通过A2A WebSocket并行查询匹配商户
- 每个商户返回最多5个相关商品
- 总查询时间 < 5秒（超时商户标记为"未响应"）
- 支持3个商户同时查询

---

### US-005: 查看聚合结果
**作为** 消费者  
**我希望** 在一个界面查看所有商户的商品结果  
**以便** 方便比较和选择  

**验收标准**:
- 商品按价格/相关度排序
- 显示商品信息：名称、价格、商户名
- 显示商品来源商户
- 支持价格筛选和排序

---

### US-006: 发起购买
**作为** 消费者  
**我希望** 点击商品即可跳转到商户完成购买  
**以便** 快速完成交易  

**验收标准**:
- 点击商品跳转到商户购买页
- 传递商品ID和参数
- 记录转化数据（用于后续优化）

## 3. MVP范围定义

### 3.1 包含功能 (MVP Core)

| 优先级 | 功能模块 | 功能描述 | 备注 |
|:------:|:--------|:---------|:-----|
| P0 | 自然语言输入 | 接收用户文本输入，支持中文 | 基础能力 |
| P0 | 意图提取 | 提取商品类目、价格区间 | 规则+NLP混合 |
| P0 | Hub商户搜索 | 调用Hub API搜索匹配商户 | 依赖Hub端 |
| P0 | A2A并行查询 | WebSocket并行查询商户商品 | 最多3商户并发 |
| P0 | 结果展示 | 列表展示商品名称、价格、商户 | 基础UI |
| P1 | 参数确认 | 向用户确认理解结果 | 降低误差 |
| P1 | 排序筛选 | 按价格排序、价格区间筛选 | 提升体验 |
| P1 | 跳转购买 | 点击商品跳转商户页面 | 转化闭环 |

### 3.2 排除功能 (Post-MVP)

| 功能 | 原因 | 计划版本 |
|:-----|:-----|:---------|
| 用户登录/注册 | 增加复杂度，MVP支持匿名使用 | v0.2 |
| 历史记录 | 需要用户系统支撑 | v0.2 |
| 收藏功能 | 非核心流程 | v0.3 |
| 智能推荐 | 需要数据积累 | v0.3 |
| 多轮对话 | 增加复杂度 | v0.2 |
| 图片搜索 | 技术复杂度较高 | v0.4 |
| 聊天式交互 | 先验证核心流程 | v0.2 |
| 订单追踪 | 依赖商户端能力 | v0.3 |

### 3.3 技术约束

```yaml
性能指标:
  意图解析响应: < 2s
  Hub搜索响应: < 3s
  A2A并行查询: < 5s (含超时)
  总流程时间: < 10s

并发限制:
  单次查询商户数: 最多3个
  每商户返回商品: 最多5个
  WebSocket超时: 5s

错误处理:
  Hub无响应: 提示"服务暂时不可用"
  商户无响应: 标记"未响应"，显示其他商户结果
  无匹配商户: 提示"未找到相关商户，尝试其他关键词"
  无匹配商品: 提示"该价位暂无商品，扩大价格范围？"
```

## 4. 数据模型

### 4.1 用户意图 (UserIntent)

```typescript
{
  rawInput: string;           // 原始输入
  category: string;           // 商品类目
  priceMin?: number;          // 最低价格
  priceMax?: number;          // 最高价格
  extractedAt: timestamp;     // 提取时间
  confidence: number;         // 置信度 0-1
}
```

### 4.2 商户信息 (Merchant)

```typescript
{
  merchantId: string;         // 商户ID
  name: string;               // 商户名称
  tags: string[];             // 标签列表
  a2aEndpoint: string;        // A2A WebSocket地址
  status: 'active' | 'offline';
}
```

### 4.3 商品信息 (Product)

```typescript
{
  productId: string;          // 商品ID
  merchantId: string;         // 所属商户
  name: string;               // 商品名称
  price: number;              // 价格
  currency: string;           // 货币 (CNY)
  category: string;           // 类目
  url: string;                // 购买链接
  image?: string;             // 商品图片URL (可选)
}
```

## 5. API接口定义

### 5.1 内部接口 (用户端 ↔ Hub)

```http
POST /api/hub/search
Content-Type: application/json

Request:
{
  "category": "茶杯",
  "tags": ["茶具"],
  "limit": 3
}

Response:
{
  "merchants": [
    {
      "merchantId": "m001",
      "name": "茶韵轩",
      "a2aEndpoint": "wss://merchant1.io011.com/a2a"
    }
  ]
}
```

### 5.2 A2A接口 (用户端 ↔ 商户端)

```http
WebSocket: wss://{merchant}/a2a

Query Message:
{
  "type": "product_query",
  "queryId": "q123",
  "intent": {
    "category": "茶杯",
    "priceMin": 100,
    "priceMax": 200
  },
  "limit": 5
}

Response Message:
{
  "type": "product_result",
  "queryId": "q123",
  "products": [
    {
      "productId": "p001",
      "name": "紫砂茶杯",
      "price": 128,
      "url": "https://..."
    }
  ]
}
```

## 6. 用户交互流程

### 6.1 主流程

```
用户: 我想买茶杯，100-200元

系统: [意图提取] 类目:茶杯, 价格:100-200元
系统: 为您搜索100-200元的茶杯，对吗？

用户: 对的

系统: [调用Hub搜索] 找到3家商户...
系统: [并行A2A查询] 查询商户A、B、C...

系统: 为您找到以下商品：
      ┌─────────────────────────────────────┐
      │ 1. 紫砂茶杯 ¥128 - 茶韵轩          │
      │ 2. 陶瓷茶杯 ¥156 - 茗香阁          │
      │ 3. 玻璃茶杯 ¥89  - 清茶馆 (超出预算)│
      │ ...                                │
      └─────────────────────────────────────┘
      点击商品查看详情并购买

用户: [点击商品1]

系统: [跳转] https://merchant1.io011.com/product/p001
```

### 6.2 异常流程

**场景1: Hub无响应**
```
系统: 服务暂时不可用，请稍后再试
```

**场景2: 商户无响应**
```
系统: 茶韵轩响应超时，为您显示其他商户商品...
```

**场景3: 无匹配结果**
```
系统: 未找到100-200元的茶杯，建议：
      1. 扩大价格范围到50-300元？
      2. 尝试搜索其他类目？
```

## 8. 风险与缓解

| 风险 | 影响 | 缓解措施 |
|:-----|:-----|:---------|
| 意图提取不准确 | 高 | 添加确认环节，允许用户修正 |
| 商户响应慢 | 中 | 设置5秒超时，异步展示 |
| Hub API故障 | 高 | 本地缓存热点商户列表 |
| 商户商品数据不一致 | 低 | 展示"价格以商户页为准"提示 |
| 并发查询性能问题 | 中 | 限制3商户并发，优化超时逻辑 |
